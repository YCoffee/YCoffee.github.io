Tomcat安装与启动运行

CoffeeY 发表于 2019-09-10

Linux下安装与启动运行

此文以Centos 7为例,Tomcat为9.0.24

1、下载Tomcat Croe压缩包（文章下载tar.gz格式）

官网：https://tomcat.apache.org

2、解压压缩包

[root@Tomcat ~]# tar -xzvf apache-tomcat-9.0.24.tar.gz

3、配置环境变量

方式一：

1、进入解压缩后目录并进入bin文件夹，打开catalina.sh，在

.# OS specific support. $var must be set to either true or false.

语句上面添加：

CATALINA_HOME=/YourTomcatPath/apache-tomcat-9.0.24(YourTomcatPath替换为你自己的Tomcat解压目录)
JAVA_HOME=/YourJdkPath/jdk1.8.0_121
JRE_HOME=$JAVA_HOME/jre
方式二：

1、编辑系统环境配置文件

[root@Tomcat ~]# vi /etc/profile

2、在最后添加JAVA_HOME、CATALINA_HOME、JRE_HOME并修改PATH

方式三：

此方式实则为方式一和方式二的混合配置，对应的三个参数混合在相应文件进行配置

4、启动Tomcat

1、进入解压Tomcat的bin目录

[root@Tomcat ~]# cd /YourTomcatPath/apache-tomcat-9.0.24/bin

2、执行./startup.sh即可

[root@Tomcat ~]# ./startup.sh

Windows下安装与启动运行

此文以Windows 10专业版64-bit为例,Tomcat为9.0.21

1、下载Tomcat Croe压缩包（文章下载格式）

官网：https://tomcat.apache.org

2、解压压缩包

3、配置环境变量

CATALINA_HOME=/YourTomcatPath/apache-tomcat-9.0.21
CATALINA_BASE=/YourTomcatPath/apache-tomcat-9.0.21
JAVA_HOME=/YourJdkPath/jdk1.8.0_121
JRE_HOME=/YourJrePath/jre1.8.0_121

4、配置系统服务

1、Win+R，键入cmd并回车
2、进入解压Tomcat的bin目录


3、执行service.bat install

5、启动Tomcat

方式一：

进入解压Tomcat的bin目录，运行startup.bat

方式二：

进入解压Tomcat的bin目录，运行tomcat9w.exe，界面内点击start按钮

方式三：

右键点击桌面上的“此电脑”—>“管理”—>“服务和应用程序”->“服务”，找到“Apache Tomcat 9.0 Tomcat9”服务，右键点击该服务，选择“启动”。
